# 🚍 Sistema de Gestión de Vehículos (Práctica CLI)

Proyecto práctico desarrollado en **JavaScript** como ejercicio de aprendizaje.  
Simula un sistema interno para la administración de flota de una empresa de transporte (buses y furgones).

El sistema funciona completamente desde consola (**CLI - Command Line Interface**), sin interfaz gráfica.

---

## 📌 Descripción

Este proyecto tiene como objetivo aplicar lógica de programación y conceptos fundamentales de JavaScript en un entorno basado en un caso real empresarial.

Permite gestionar información relacionada con:

- Vehículos
- Mantenciones
- Gastos operativos
- Historial mecánico
- Reportes básicos

---

## 🎯 Objetivos de la práctica

- Practicar estructuras de datos.
- Aplicar Programación Orientada a Objetos.
- Manipular arrays con métodos avanzados.
- Simular un sistema administrativo real.
- Organizar código de forma modular.

---

## ⚙️ Funcionalidades

- ✔ Registrar vehículos (patente, tipo, kilometraje, estado).
- ✔ Registrar mantenciones y servicios mecánicos.
- ✔ Registrar gastos (combustible, repuestos, reparaciones).
- ✔ Consultar historial por vehículo.
- ✔ Generar reportes simples desde consola.

---

## 🛠 Tecnologías utilizadas

- JavaScript
- Node.js
- Consola (CLI)
